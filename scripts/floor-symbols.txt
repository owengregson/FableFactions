# FableFactions floor-symbol manifest (AM-13)

# The committed floor for the two in-process ASM build gates:
#   verifyDescriptorFloor   — a class registered as a BASELINE (non-@ProbeGated)
#                             org.bukkit.event.Listener must not mention any
#                             [POST_FLOOR_TYPES] entry in a method/field descriptor
#                             (Bukkit's createRegisteredListeners reflects EVERY
#                             declared member; one absent type registers ZERO handlers
#                             and silently kills that whole listener — GAP 1).
#   verifyNoStickyGetstatic — no GETSTATIC of any [POST_FLOOR_ENUM_CONSTANTS] entry
#                             (a direct getstatic of a sub-floor enum constant is a
#                             STICKY NoSuchFieldError re-thrown on every event — GAP 2);
#                             such constants must flow through the Constants resolver
#                             / Enum.valueOf.
#
# Format: one JVM-internal symbol per line; '#' starts a comment; blank lines ignored.
#   [POST_FLOOR_TYPES]           internal class names  (org/bukkit/NamespacedKey)
#   [POST_FLOOR_ENUM_CONSTANTS]  <ownerInternalName>.<CONSTANT>
#
# UPGRADE PATH (AM-13): this hand-curated hazard list — sourced from version-deltas.md
# §3.8 / Part 1 (every known post-1.7.10 type, member and enum constant a factions
